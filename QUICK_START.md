# 🚀 Quick Start Guide - Ethiopia BookStore

## Step-by-Step Setup (5 Minutes)

### Step 1: Setup Database (1 minute)
Open in browser:
```
http://localhost/online_bookstore/setup.php
```
✓ Creates database
✓ Creates all tables
✓ Adds admin account
✓ Adds sample categories

### Step 2: Add Ethiopian Books (1 minute)
```
http://localhost/online_bookstore/add_ethiopian_books.php
```
✓ Adds 30+ Ethiopian academic books
✓ Adds Ethiopian fiction books
✓ Adds children's books
✓ All books ready to sell!

### Step 3: Test Everything (1 minute)
```
http://localhost/online_bookstore/test_database.php
```
✓ Verify all tables exist
✓ Check books are loaded
✓ Confirm admin account

### Step 4: Access Application (2 minutes)

#### Homepage
```
http://localhost/online_bookstore/index.php
```
- Browse Ethiopian books
- Search and filter
- View book details

#### Admin Panel
```
http://localhost/online_bookstore/admin/login.php
Username: admin
Password: admin123
```
- Manage products
- Manage orders
- View statistics

#### Student/Customer
```
http://localhost/online_bookstore/student/register.php
```
- Register new account
- Login and shop
- Place orders

## 🎯 Complete User Flow Test

### Test as Student:
1. **Register**: Create account with email/password
2. **Browse**: View Ethiopian books on homepage
3. **Filter**: Try category filters (Academic, Fiction, etc.)
4. **Add to Cart**: Click "Add to Cart" on any book
5. **View Cart**: Check cart shows items
6. **Add Address**: Go to "Manage Addresses" and add shipping address
7. **Checkout**: Proceed to checkout, select address
8. **Place Order**: Complete order with COD payment
9. **View Orders**: Check "My Orders" to see order status

### Test as Admin:
1. **Login**: Use admin/admin123
2. **Dashboard**: View statistics
3. **Manage Products**: Add/edit/delete books
